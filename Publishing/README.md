## node-sass 사용방법

### cd Publishing <= 퍼블리싱 폴더로 이동

### 특정 파일을 특정 파일 이름으로 컴파일

### Compile foo.scss to bar.css

### node-sass foo.scss > bar.css

### 폴더 내의 모든 파일을 컴파일

### node-sass input-folder-path -o output-folder-path

### node-sass src/sass --output dist/css
